# Mete Güneş - Brand Guidelines

## Logo

### Primary Logo
- SVG logo sidebar'da kullanılır
- Dark background (#1a1a1a) üzerinde cream text (#e8e4df)
- Minimum boyut: 120px genişlik

### Favicon
- 32x32 SVG (modern tarayıcılar)
- PNG fallback (legacy)

---

## Color Palette

### Primary Colors

| Renk | Hex | Kullanım |
|------|-----|----------|
| **Ink** | `#1a1a1a` | Text, backgrounds, logo |
| **Cream** | `#f5f1eb` | Background, surfaces |
| **White** | `#ffffff` | Cards, clean surfaces |

### Accent Color

| Renk | Hex | Kullanım |
|------|-----|----------|
| **Blue** | `#3b82f6` | CTAs, links, focus states |
| **Blue Light** | `#60a5fa` | Hover states |
| **Blue Dark** | `#2563eb` | Active states |

### Text Colors

| Renk | Hex | Kullanım |
|------|-----|----------|
| **Primary** | `#1a1a1a` | Headings, body |
| **Secondary** | `#555555` | Subtext |
| **Muted** | `#888888` | Captions, placeholders |

## Typography

### Font Family
- **Primary:** Inter (Google Fonts)
- **Display:** Space Grotesk (headings)
- **Fallback:** -apple-system, BlinkMacSystemFont, sans-serif

### Font Sizes
```
Hero:     64px / 700
H1:       52px / 700
H2:       32px / 600
H3:       24px / 600
Body:     16px / 400
Small:    14px / 400
Caption:  12px / 500
```

## Voice & Tone

- **Profesyonel** ama samimi
- **Net** ve özlü
- **Güven** verici
- Teknik detayları **basit** anlat

---

## Usage Guidelines

### Do's ✅
- Logo etrafında yeterli boşluk bırak
- Brand renklerini tutarlı kullan
- High contrast text/background
- Responsive font sizes

### Don'ts ❌
- Logo'yu deforme etme
- Brand renkleri dışında renkler kullanma
- Low contrast combinations
- Çok fazla font weight karışımı
